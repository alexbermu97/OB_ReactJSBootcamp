Posicionamiento en CSS

  • Static: el formato estándar.
  • Relative: formato relativo a la posición estática.
  • Fixed: formato fijo para elementos que no se mueven.
  • Absolute: similar al formato fijo pero en relación a un elemento padre. Se suele usar en elementos que van dentro de otros, con el otro siendo relativo.
  • Sticky: mezcla entre relativo y fijo. Común en menús y otros elementos que queremos que se vean cuando el usuario scrolea a partir de cierto punto.
